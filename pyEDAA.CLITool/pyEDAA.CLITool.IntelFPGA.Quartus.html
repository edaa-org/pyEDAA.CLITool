pyEDAA.CLITool.IntelFPGA.Quartus

This module contains the CLI abstraction layer for Quartus.

Classes

  • Map: Represent a CLI executable derived from Program, that adds an abstraction of subprocess.Popen.

  • TclShell: Represent a CLI executable derived from Program, that adds an abstraction of subprocess.Popen.
