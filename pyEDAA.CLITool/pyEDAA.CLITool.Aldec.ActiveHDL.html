pyEDAA.CLITool.Aldec.ActiveHDL

This module contains the CLI abstraction layer for Active-HDL.

Classes

  • VHDLLibraryTool: Abstraction of Active-HDL’s VHDL library management tool vlib.

  • VHDLCompiler: Abstraction of Active-HDL’s VHDL compiler vcom.

  • VHDLSimulator: Abstraction of Active-HDL’s simulator vsim.
