pyEDAA.CLITool.Xilinx.Vivado

This module contains the CLI abstraction layer for Vivado.

Classes

  • Vvgl: Represent a CLI executable derived from Program, that adds an abstraction of subprocess.Popen.

  • XElab: Represent a CLI executable derived from Program, that adds an abstraction of subprocess.Popen.

  • XSim: Represent a CLI executable derived from Program, that adds an abstraction of subprocess.Popen.

  • Synth: Represent a CLI executable derived from Program, that adds an abstraction of subprocess.Popen.
